## Known Limitations (Documented)

### 1. macOS 26.1 Beta Test Failures
- **Limitation**: 9 tests fail on macOS 26.1 beta
- **Cause**: NWListener binding OS bug
- **Severity**: Low (beta OS only)
- **Workaround**: Use stable macOS release
- **Documentation**: README, CHANGELOG, INTEGRATION.md

### 2. No STARTTLS/TLS Support
- **Limitation**: No encryption support in v0.1.0
- **Severity**: Medium (expected for v0.1.0)
- **Workaround**: Use reverse proxy with TLS termination
- **Documentation**: CHANGELOG, README
- **Plan**: Consider for v0.2.0

### 3. No SMTP AUTH
- **Limitation**: No authentication in v0.1.0
- **Severity**: Medium (expected for v0.1.0)
- **Workaround**: Deploy on trusted networks only
- **Documentation**: CHANGELOG, README
- **Plan**: Consider for v0.2.0

### 4. No Rate Limiting
- **Limitation**: No built-in rate limiting
- **Severity**: Low
- **Workaround**: Implement in message handler or use external tools
- **Documentation**: INTEGRATION.md
- **Plan**: Consider for future release
